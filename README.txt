Setup steps:

1. Setup openCV for ubuntu:
    a. `sudo apt-get install python-opencv`
2. Create virtualenv [in vframe_pdf directory]
    a. `python3 -m pip install --user virtualenv` #install virtualenv
    b. `python3 -m venv env` #create virtualenv for this directory
    c. `source env/bin/activate` #activate the virtualenv
    d. `pip install -r requirements.txt`
